Import Historical Data ? How to import historical data

1. Copy your Max/MSP CSV files into the data/import/ folder inside this project directory.

2. Click Scan for files to see what's available.

3. Fill in the participant details and select the main pressure file (30 s). For Test 1, you can also assign up to 3 calibration files (10 s each).

4. Tick Mark as complete immediately if the test type has no questionnaire steps.

5. Click Import Session — files are copied into the session folder and a record is created in the database.
